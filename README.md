# ENGEAR Quotation Builder

ระบบสร้าง BOQ / Quotation สำหรับงานตกแต่งภายใน

## Features
- 📋 BOQ Builder — สร้างรายการงานแบบ hierarchical
- 💰 Unit Price Database — ฐานข้อมูลราคามาตรฐาน
- 🔄 Revision History — ติดตามทุก version
- 📊 ค่าใช้จ่ายเพิ่มเติม — OT, เดินทาง, ขอเข้าพื้นที่
- 📥 Export Excel — ออก Quotation + BOQ + Revision log
- 🖨 Print — พิมพ์เป็น A4 format

## วิธีใช้
เปิด `index.html` ในเว็บ browser ได้เลย ไม่ต้อง install อะไร

## Tech
- Single HTML file (Vanilla JS + CSS)
- XLSX.js สำหรับ Export Excel
- ไม่มี dependencies / framework
